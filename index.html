Firewall

Tipos de Firewall – Uma Firewall é simplesmente um programa ou um despositivo de hardware que filtra a informação que nos chega através da ligação à Internet, até ao nosso computador ou rede privada. No entanto existem vários tipos de Firewall que se adequam a diferente cenários, como por exemplo, estrutura de rede, necessidades específicas do utilizador ou características do sistema opperativo. Quais os tipos de Firewall mais importantes? Filtros de Pacotes (Pack Filtering): Este tipo de Firewall utilizam regras para filtrarem pacotes. É um método simples mas oferece algum nível de segurança. A filtragem de pacotes analisa cada pacote que passa pela rede, aceitando ou não, tendo por base as regras determinadas por o utilizador. Este tipo de Firewall possui um maior desempenho comparado aos outros tipos de Firewall. Destaca-se por ser uma Firewall de baixo custo. simples e flexível. No entanto a filtragem de pacotes tem um nível de segurança bastante primário, por isso mesmo é aconselhável o uso de firewall por software (Firewall do Windows) como complemento, melhorando assim a segurança da rede. A filtragem de pacotes é normalmente feita a pacotes IP e baseada nos seguintes campos: Endereço IP fonte, Endereço IP destino, Porta TCP/UDP fonte e Porta TCP/UDP destino. Proxy (Firewall de aplicação): Este tipo de Firewall capta todas a informação que entra e sai da rede. É o intermediário entre a rede externa e a rede interna, não permitindo uma comunicação directa entre ambos. Todo o tráfego que passa a fronteira entre a parte pública e a parte privada é encaminhado para a proxy, que funciona a nível de aplicação, este tipo de firewall também possibilita a autenticação do usuário e o controlo de estado das ligações, permitindo assim uma filtragem mais eficaz da informação. Deixa de existir acesso direto do sistema externo ao sistema interno, com este processo a segurança é reforçada. Filtros baseado no estado (stateful inspection): Este tipo de Firewall é uma evolução do filtro de pacote. Esta faz um género de comparação entre o que está a acontecer e o que é esperado que aconteça, analisando todo o tráfego de dados procurando padrões aceitáveis tendo em conta as regras estabelecidas. Este tipo de Firewall analisa o tráfego, constuindo tabelas de estado das ligações à Internet. Permite usar a junção de packet filter e proxy. O preço elevado desta tecnologia poderá ser um factor contra mas se o seu principal objectivo é a segurança, então esta será a melhor solução. É uma tecnologia muito mais eficiente que as anteriores e com muitos menos necessidade de intervenção.
